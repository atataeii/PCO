# Personal Portfolio Website

This project was created in **2024** as part of the **PCO course** at [HZ University of Applied Sciences](https://hz.nl).

The goal of this assignment was to design and build a personal portfolio website using **HTML**, **CSS**.

---

## 📌 Project Overview

This website was developed as one of my first front-end projects. It introduces me as an ICT student and presents information about my studies, motivation, goals, and academic progress.

The website consists of the following pages:

- Home
- Profile
- Dashboard
- Blog
- FAQ
